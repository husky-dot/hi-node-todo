# 基于 node 写一个 todo 命令行工具

**功能**
- 列出所有的 todo
- 新增 todo
- 编辑 todo
- 删除 todo
- 标记 todo 为已完成/未完成



## 安装
```
yarn add hi-node-todo
```

## 使用


**显示任务**
```
t  list  
```

![](https://klxxcdn.oss-cn-hangzhou.aliyuncs.com/histudy/hrm/media/66/node1.png)


**添加任务**
```
t add 看书  
```

![](https://klxxcdn.oss-cn-hangzhou.aliyuncs.com/histudy/hrm/media/66/node2.png)


**清除所有任务**
```
t clear
```
